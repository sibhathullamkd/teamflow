ℹ️ Capabilities & Limitations

✓ What this admin panel CAN do:

• View all user accounts with creation date and last login time

• Read, search, and delete any user's notes, tasks, and events

• View login session history (last sign-in timestamps from Supabase auth)

• Send password reset emails via Supabase (use Supabase Dashboard)

• Delete all data for a specific user

• Log admin actions locally in the Activity Log


✗ Limitations (frontend-only app):

• Password reset — Cannot directly change passwords without the Supabase service role key. Go to Supabase Dashboard → Authentication → Users → ⋮ → Send Recovery Email

• Email change — Same limitation. Must be done via Supabase Dashboard

• Real IP logging — Browsers cannot log server-side IPs. Only Supabase's built-in metadata (last sign-in time) is available

• Device tracking — Full device fingerprinting requires a backend service

• Real-time presence — Cannot see who is currently online without a WebSocket backend
